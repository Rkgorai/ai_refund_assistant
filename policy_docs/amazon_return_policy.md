# E-Commerce Return & Refund Policy

This document outlines the strict return and refund guidelines for all items purchased through our platform. All customer support agents (human and AI) must strictly enforce these rules.

## General Return Window

The standard return window depends on the item category and the specific `return_policy` noted on the item. **The return window always starts from the `delivery_date`**, NOT the `order_date`.

If an item's `delivery_date` is outside the specified `return_policy` window, the return request **MUST BE DENIED**, regardless of the customer's circumstances.

## Specific Return Policies

### 1. "7 Days Return" or "15 Days Return" or "30 Days Return"
Items marked with these policies are eligible for a full refund if the request is made within the specified number of days from the `delivery_date`. 
- Valid reasons for return: Defective, damaged upon arrival, incorrect item, or no longer needed.
- Must be returned in original packaging.

### 2. "7 Days Replacement" (or any Replacement Only policy)
Items marked as "Replacement Only" are **NOT ELIGIBLE FOR A REFUND** under any circumstances. 
- If the item is defective or damaged, the customer may request a *Replacement* within the specified window (e.g., 7 days).
- The AI agent should inform the customer that a refund cannot be processed, but a replacement will be shipped immediately.

### 3. "Non-Returnable"
Items marked as "Non-Returnable" cannot be returned, refunded, or replaced. 
- Examples usually include: Grocery items, hygiene products (e.g., undergarments, toothbrushes), digital goods (e.g., gift cards, e-books, digital codes).
- The AI agent MUST immediately deny refund requests for these items, citing the "Non-Returnable" policy for hygiene, digital, or perishable goods.

## Refund Processing Timeline

If an item is eligible for a refund and the return is processed:
- **Credit Card / Debit Card**: 3-5 business days after the return is received.
- **UPI**: 1-2 business days.
- **Store Credit**: Instant, as soon as the item is dropped off.

## Edge Cases & Escalations
- If a customer claims a "Non-Returnable" item (e.g., Groceries) arrived completely spoiled or hazardous, the AI agent must NOT process a refund directly. Instead, the agent must **escalate** the issue by creating a Support Ticket for human review.
- If a customer is abusive or uses profanity when a refund is denied, the AI agent should politely end the conversation and log the ticket as 'Closed'.
